# **taiki-cli**

taiki-cli 是构建 typescript 应用的超级助手。

## 🚀 使用 

> 🪐 🦆 🦮  🐕 🍑 🥜 ☕️ 🍹 🧉 🍾 🧊 🥅 🏆 🥇 🎲 🚲 ✈️ 🗽 🗼 🚧 💵 💴 🪤 🧱 💣 🧨 🪓 📖 🔖 📍 ✂️ 📝 ✏️ 🔒 🔓
🇨🇳 🏴‍☠️ 🤯 🧠 👨‍🎓 ☂️ 🧵 👠 💄 🛀🏻 🐳 🍼 🪁 🏹

直接创建项目（不安装）

```sh
npx taiki-cli <path-to-project>
```

或者：

1. 全局安装

```sh
pnpm add taiki-cli -g
# or
yarn global add taiki-cli
# or
npm install taiki-cli -g
```

2. 创建项目

```sh
taiki-cli <path-to-project>
```
